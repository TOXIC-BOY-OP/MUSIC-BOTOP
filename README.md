<h2 align="center">
   Install Locally Or On A VPS
</h2>

We Suggest To use Debian v-11 else ubuntu v-20+
```console
$ git clone https://github.com/TOXIC-BOY-OP/MUSIC-BOTOP
$ cd MUSIC-BOTOP
$ bash setup
$ cp sample.env .env
```

<h3 align="center">
    Edit <b>.env</b> with your own values and Run Bot
</h3>

```console
$ screen
$ python3 -m Zaid
```
